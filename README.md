# xdv-draft

## Non-Normative Archive (Historical Drafts)

Status: **Archived / Non-Normative**  
Canonical specification source: `../xdv-spec`  
Scope: historical draft material retained for traceability only.

## Archive Policy

1. `xdv-draft` is **not** authoritative for XDV requirements.
2. Normative requirements, status tags, and pass/fail criteria are defined in `xdv-spec` only.
3. New technical work, requirement edits, and architecture changes must be authored in `xdv-spec`.
4. Changes in `xdv-draft` are limited to archival metadata, cross-links, and historical notes.

## Drift Control

To prevent drift between draft text and normative spec:

- Do not add or modify requirement semantics in this directory.
- Treat every technical statement here as historical context only.
- If a mismatch is found, `xdv-spec` is the source of truth.
- Open/update items in `xdv-spec` first; optionally add a historical note here after the canonical update.
